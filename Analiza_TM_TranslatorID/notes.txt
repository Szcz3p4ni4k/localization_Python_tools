- przykładowy początkowy tag <tu> : <tu changedate="20200407T181713Z" creationdate="20200407T181438Z" creationid="nkrakowska" changeid="nkrakowska">
- nazwa TMa/ TMów gdzie tłumacz ostatni raz wprowadzał zmiany - creation i change ID - id tłumacza i kiedy ostatni raz prowadzał zmiany
- analiza TMów zapisana w pliku CSV
- kolumny pliku CSV:
	- nazwa TMa | ID Tłumacza | data ostatniego segmentu | data ostatniej zmiany | ilość stworzonych segmentów | ilość zmienionych segmentów  | ilość stworzonych znaków | ilość zmienionych znaków
	- jeżeli brakuje informacji do kolumny wstaw "-"
	- nazwa TMa - nazwa pliku
	- ID Tłumacza - pobranie informacji z creationid i changeid połączenie ich i usunięcie duplikatów
	- data ostatniego segmentu - pobranie informacji z creationdate="20150825T100013Z" - ostatnie wystąpienie creationdate dla danego creationid
	- data ostatniej zmiany - pobranie informacji z changedate="20150825T100013Z"- ostatnie wystąpienie changedate dla danego changeid. Jeżeli changedate = creationdate i creationid = changeid nie bierzemy tego segmentu pod uwagę do wartości w data ostatniej zmiany
	- ilość utworzonych segmentów - ilość wystąpień creationid w TMie dla danego id tłumacza
	- ilość zmienionych segmentów - ilość wystąpień changeid w TMie dla danego id tłumacza. Jeżeli changedate = creationdate i creationid = changeid nie brać pod uwagę tego segmentu do ilości ilość zmienionych segmentów, liczymy wtedy tą informację tylko do ilość stworzonych segmentów
	- Liczenie znaków:
		- ilość stworzonych znaków - ilość znaków pomiędzy tagami targetowymi ( <tuv xml:lang="en"> </tuv>) z uwzględnieniem targetu TM <prop type="targetlang">en</prop>. Bierzemy tekst pomiędzy tagami razem ze spacjami. Suma znaków w danym TMie dla danego tłumacza (creationid)
		- ilość zmienionych znaków - ilość znaków pomiędzy tagami targetowymi ( <tuv xml:lang="en"> </tuv>) z uwzględnieniem targetu TM <prop type="targetlang">en</prop>. Bierzemy tekst pomiędzy tagami razem ze spacjami. Suma znaków w danym TMie dla danego tłumacza (changeid)
		- Jeżeli changedate = creationdate i creationid = changeid nie brać pod uwagę tego segmentu do ilości zmienionych znaków, liczymy wtedy tą informację tylko do ilość stworzonych znaków
- informacja na temat kodowania pliku .tmx:
		<?xml version="1.0" encoding="utf-16"?>
		<!DOCTYPE tmx SYSTEM "tmx14.dtd">
		<tmx version="1.4">
